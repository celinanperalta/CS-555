# CS-555 GEDCOM Reader
## Getting Started
Install pipenv: https://pipenv.pypa.io/en/latest/install/#:~:text=.-,%E2%98%A4%20Installing%20Pipenv

Run the following:
```
cd gedcom
pipenv install
```

## How to Run
Note: X = project number i.e. gedcom/project03
```
cd gedcom/project0X
chmod u+x main.py
python main.py --file <filename>
```
